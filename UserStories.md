# User Stories & Aufgaben

## User Story 1
Als Nutzer möchte ich Aufgaben erstellen können, damit ich meine Lernziele organisieren kann.

### Aufgaben
- Eingabefeld für neue Aufgaben erstellen
- Button zum Hinzufügen implementieren
- Aufgabenliste anzeigen
- Aufgaben lokal speichern

---

## User Story 2
Als Nutzer möchte ich Aufgaben als erledigt markieren können, damit ich meinen Fortschritt sehen kann.

### Aufgaben
- Checkbox für Aufgaben hinzufügen
- Status „erledigt“ speichern
- Erledigte Aufgaben visuell hervorheben
- Fortschrittsanzeige erstellen

---

## User Story 3
Als Nutzer möchte ich einen Fokus-Timer verwenden können, damit ich konzentrierter lernen kann.

### Aufgaben
- Timer-Komponente erstellen
- Start-/Pause-Button hinzufügen
- Countdown implementieren
- Benachrichtigung nach Ablauf anzeigen
